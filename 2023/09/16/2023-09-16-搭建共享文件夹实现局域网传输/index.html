共享文件夹的建立

条件：同一局域网（在同一个WiFi，同一热点）

一、电脑操作

1. 新建文件夹
2. 设置文件夹的属性，选着共享
3. 设置高级共享，权限选择完全控制
4. 选择共享，选择Everyone
5. 在windows搜索栏中输入cmd，打开命令提示符窗口
6. 输入ipconfig回车
7. 记住你的ipv4地址

二、ipad操作

1. 打开iPad自带的文件管理器
2. 选择连接服务器
3. 输入服务器名称 smb://192.1XX.XX.XXX  连接
4. 选择注册用户
5. 输入window计算机的账号密码（注意：密码不是pin码）

三、注意

1. iPad和电脑必须连接同一个wifi
2.断开后再次连接wifi可以直接连接，不需要再次重复上述操作。

​

①输入的服务器用户名或密码错误
②连接服务器时出现问题
先把电脑名称和账户名称都改成纯英文，还不行的看下面:
一、检查网络设置
设置–网络和internet–状态–属性–网络配置文件选择专用
二、检查smb协议
win+r输入control 回车–程序–启用或关闭windows功能–勾选SMB1.0/CIFS文件共享支持
三、检查本地策略
win+r输入secpol.msc 回车–本地策略–安全选项–网络访问:本地账户的共享和安全模型–右键属性选择经典

或者直接不设置密码好了

一、设置网络安全共享

1、打开网络共享中心

2、更改高级共享设置

3、全选

tips：也不一定要全选，只要打开网络发现然后不要设置密码就可以了（设置了很有可能其他设备连接时不知道用户名）

二、任意新建一个文件夹，在属性里打开共享

在权限里设置，添加everyone，检查名称后就出现了，然后将其权限给满

3、用win + r打开命令行输入cmd，在其中输入ipconfig，也可直接在设置里查看ip地址

再次 win + r

输入\\+ip地址

将之前创建的连接拖入此电脑方便打开（不拖入也是没问题的哈）

smb协议，可以在unix linux WINDOWS mac iOS之间互传文件，如果有接触软路由和群晖的人对这个会比较了解，我家里就是直接拿路由器（软）接1t移动硬盘，做一个小型的服务器存储，凡是连接到这个路由器的，都可以自由的访问移动硬盘里的文件，而且不会对操作系统挑三拣四，对动手能力的要求比较高，这种方法取决于局域网的传输速度，比如我家里车往传输速度是1000兆，那么每秒传输速度的理论上限在256M左右，好处是对于文件类型大小都没有任何限制，想学点技术的，可以百度搜索恩山论坛，多爬爬帖子就行了
