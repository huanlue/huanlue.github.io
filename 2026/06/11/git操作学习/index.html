第一步: 设置ssh的公钥和私钥

生成新的 SSH 密钥并将其添加到 ssh-agent。
ssh key-gen命令生成公钥和私钥

代码参数含义：
-t 指定密钥类型，默认是 rsa ，ed25519。
-C 设置注释文字，比如邮箱。
-f 指定密钥文件存储文件名。

执行命令后会生成两个文件公钥li4git.pub, 私钥li4git.lab.
密钥过多可以设置config文件进行调整.
然后将公钥.pub的所有内容上传到github个人中ssh和gpg钥匙的配置中.

这样可以访问自己的私人仓库(private)
